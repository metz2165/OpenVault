# Monthly Review

2-4 hour strategic review. Schema formation — identify patterns, update mental models.

## Checklist

### Reflect
- [ ] Read all [[110-Logs/Weekly]] entries for the month
- [ ] Identify recurring themes, wins, and friction points
- [ ] Review all decisions made in [[110-Logs/Decisions]] — any to revisit?

### Knowledge
- [ ] Create or update [[090-Knowledge/MOCs]] for emerging topics
- [ ] Prune orphaned notes — link them or archive them
- [ ] Review [[080-Bookmarks]] — remove dead links, promote useful ones to knowledge

### Projects
- [ ] Review [[010-Projects/Active]] — should any move to On-Hold or Completed?
- [ ] Review [[010-Projects/On-Hold]] — should any reactivate?
- [ ] Update project phases and timelines

### System
- [ ] Review [[040-Skills]] and [[060-Prompts]] — any updates needed?
- [ ] Review [[070-Routines]] — are the checklists still working?
- [ ] Review [[050-MCP]] registry — any new tools or deprecations?
- [ ] Check [[120-Templates]] — any new templates needed?

### Plan
- [ ] Set next month's strategic focus
- [ ] Update [[001-Dashboard/dashboard]]
- [ ] Identify 1-3 high-leverage improvements to the vault itself

## When
Last day of the month or first day of the new month.
