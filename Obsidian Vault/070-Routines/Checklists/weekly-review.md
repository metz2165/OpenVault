# Weekly Review

30-60 minute consolidation session. Transfers short-term into long-term memory.

## Checklist

### Clear
- [ ] Inbox zero — process everything in [[000-Inbox]]
- [ ] Review all [[020-Tasks]] lists — update, prune, reprioritize
- [ ] Clear [[020-Tasks/flags]] — resolve or acknowledge each item

### Review
- [ ] Scan [[110-Logs/Daily]] for the week — extract patterns or insights
- [ ] Review each active project in [[010-Projects/Active]] — update status
- [ ] Check [[030-Agents/Logs]] — any agent issues or improvements needed?
- [ ] Review [[110-Logs/Decisions]] — any pending decisions to make?

### Consolidate
- [ ] Move reusable insights from daily logs → [[090-Knowledge]]
- [ ] Update relevant [[090-Knowledge/MOCs]] with new links
- [ ] Archive completed projects → [[010-Projects/Completed]]
- [ ] Archive stale items → [[130-Archive]]

### Plan
- [ ] Set next week's priorities in [[020-Tasks/upcoming]]
- [ ] Update [[001-Dashboard/dashboard]]
- [ ] Identify 1 high-impact item for [[020-Tasks/flags#High Impact]]

## When
End of week or start of new week. Block 30-60 min.
