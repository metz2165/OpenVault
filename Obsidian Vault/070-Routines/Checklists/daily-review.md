# Daily Review

5-minute end-of-day shutdown ritual. Consolidates working memory into storage.

## Checklist
- [ ] Process [[000-Inbox/triage]] — route or delete every item
- [ ] Review [[020-Tasks/today]] — mark done items, move incomplete to [[020-Tasks/upcoming]]
- [ ] Check [[020-Tasks/flags]] — any new blockers or risks?
- [ ] Capture notes — anything worth keeping from today → [[090-Knowledge]] or project folder
- [ ] Prep tomorrow — set 1-3 focus items in [[020-Tasks/today]]
- [ ] Update [[001-Dashboard/dashboard]] if project statuses changed

## When
End of work session, every day. Non-negotiable.
